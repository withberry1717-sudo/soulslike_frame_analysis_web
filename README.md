# アクションゲーム分析Web 完全版

GitHub Pagesでそのまま公開できる、アクションゲーム分析用Webです。

## 公開方法

1. このフォルダの中身をGitHubリポジトリに入れる
2. GitHub Pagesを有効化する
3. `index.html` がトップページになります

## フレーム表記ルール

- パリィ対象攻撃：攻撃アニメーション開始と思われる地点を0Fとして数える
- 致命攻撃：致命開始を0Fとして別カウントで数える

## 注意

80Fの接触画像は、この会話内で独立した画像が確認できなかったため、直前フレームを基準にした参照画像として入れています。正確な80Fのスクリーンショットがある場合は、`Gundyr_080F_WeaponContact_reference.jpg` を差し替えてください。
