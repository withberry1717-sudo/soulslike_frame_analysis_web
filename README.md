# アクションゲーム分析Web

GitHub Pagesでそのまま公開できる、アクションゲーム分析用Webです。

## 公開方法

1. このフォルダの中身をGitHubリポジトリに入れる
2. `assets/images/ds3_gundyr_parry/` に分析画像を入れる
3. GitHub Pagesを有効化する
4. `index.html` がトップページになります

## フレーム表記ルール

- パリィ対象攻撃：攻撃アニメーション開始と思われる地点を0Fとして数える
- 致命攻撃：致命開始を0Fとして別カウントで数える

## 分類方針

- メイン分類：ゲームタイトル別
- ページ単位：アニメーション・攻撃単位
- 補助分類：パリィ、VFX、カメラ、報酬演出などのタグ
